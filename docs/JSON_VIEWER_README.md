# JSON Viewer - Инструкция по использованию

## 🚀 Быстрый старт

### Вариант 1: Веб-интерфейс (рекомендуется)

Запустите сервер:
```bash
python json_viewer_server.py
```

Или с указанием директории и порта:
```bash
python json_viewer_server.py --dir "c:\Users\Андрюша\Documents\VmixScoreData" --port 8080
```

Откройте в браузере: **http://localhost:8080**

**Возможности:**
- ✅ Автоматическое обновление каждые 2 секунды
- ✅ Подсветка синтаксиса JSON
- ✅ Показ размера файла и времени последнего изменения
- ✅ Темная тема для комфортного просмотра
- ✅ Можно отключить автообновление

## 📋 Альтернативные варианты

### Вариант 2: VS Code / Cursor

**Встроенные возможности:**
- Откройте любой `.json` файл - автоматическая подсветка
- Нажмите `Shift+Alt+F` для форматирования
- Файлы автоматически перезагружаются при изменении

**Рекомендуемые расширения:**
- **JSON Tools** - форматирование и валидация
- **JSON Viewer** - визуализация структуры
- **Auto Refresh** - автообновление открытых файлов

### Вариант 3: Онлайн-сервисы

1. **JSONLint** (https://jsonlint.com/)
   - Вставьте JSON → валидация и форматирование

2. **JSON Crack** (https://jsoncrack.com/)
   - Визуализация JSON в виде графа

3. **JSON Formatter** (https://jsonformatter.org/)
   - Красивое форматирование с подсветкой

### Вариант 4: Специализированные программы

1. **JSON Viewer Plus** (Windows Store)
   - Бесплатное приложение для Windows
   - Поддержка больших файлов

2. **JSON Crack Desktop** (https://jsoncrack.com/)
   - Десктопная версия с автообновлением

3. **Notepad++** с плагином JSON Viewer
   - Легкий редактор с подсветкой

## 🔧 Настройка веб-интерфейса

### Изменить порт:
```bash
python json_viewer_server.py --port 9000
```

### Изменить директорию:
```bash
python json_viewer_server.py --dir "D:\MyData"
```

### Запуск в фоне (Windows):
```bash
start /B python json_viewer_server.py
```

## 🐛 Решение проблем

**Сервер не запускается:**
- Проверьте, не занят ли порт: `netstat -an | findstr "8080"`
- Используйте другой порт: `--port 8081`

**Файлы не отображаются:**
- Проверьте путь к директории: `--dir "полный\путь\к\папке"`
- Убедитесь, что файлы имеют расширение `.json`

**Автообновление не работает:**
- Нажмите кнопку "🔄 Обновить все" в интерфейсе
- Проверьте, что файлы действительно обновляются на диске
