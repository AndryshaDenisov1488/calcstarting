# Обратное проектирование: стартовый порядок с клубами

Исходный артефакт: `StartingOrderWithClubNames.rpt` (OLE, см. `RPT_ANALYSIS.md`).  
Ниже — **логическая** модель, восстановленная по имени файла и по документации ISUCalcFS в репозитории (`OSIS_FULL_INSTRUCTION.md`, `JSON_TYPES_DICTIONARY.md`).

## Группировки и агрегаты

- **Группа отчёта:** одна пара `(CAT_ID, SCP_ID)` = один подпоток записей, без подытогов.
- **Агрегаты:** нет (стартовый список).
